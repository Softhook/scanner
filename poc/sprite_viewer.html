Design Investigation Comparison

We investigated the existing Phantom modular designs to see how they can be modified to feel cuter and more aligned with Pokémon-style visual motifs.

The core issue with the Classic Set is that they are built with sharp, glitchy, and hollow shapes (like TV Static and Skull). The proposed Cute Set uses rounded silhouettes, prominent eyes with reflections (shiny pupils), sprouts/ears, chubby frames, and softer appendages (like mermaid tails and wheels).

How They Are More Distinctive

  • Silhouette Variety: Instead of simple shapes, heads vary between long pointed ears (Foxy), side gills (Axo), pointy wizard cowl (Ghosty), and leaf sprout (Sprout).
  • Clear Eye Motifs: The classic set has hollow skulls or chaotic static. The cute set has solid 2x2 pupils, winks, and highlights that give immediate personality.
  • Texture Contrast: The bodies vary between soft chubby curves, flared wings, fluffy cloud textures, and segmented checker shell lines.
  • Movement Style: The feet represent different movement types: flat stubby paws (walking), side-slithering ghost trail (hovering), double-tipped mermaid fins (swimming), and rollers (gliding).
